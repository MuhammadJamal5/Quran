Thank you for downloading the Quran Reel Generator.

HOW TO USE:
-----------
1. Double-click "Start_App.bat".
2. The application will launch and automatically open your web browser.
3. Select your Reciter, Surah, and Verses.
4. Click "Generate Reel".

FEATURES:
---------
- 100% Offline Generation
- NVIDIA GPU Acceleration Enabled
- Parallel Processing for Fast Rendering
- Professional Fade Transitions

TROUBLESHOOTING:
----------------
- If the browser does not open, go to: http://localhost:5000
- Ensure you have the "System" folder in the same directory.
